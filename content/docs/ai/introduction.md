# AI & Intelligence Layer

> **Note:** This feature is planned for a future release and is not yet implemented.

OmniSvelte aims to collapse the complexity of integrating AI by providing typed primitives out of the box.

## Planned Features

- **Vector/Embedding Column Type:** Native pgvector support with similarity search.
- **`defineAgent()`:** Typed LLM tool-calling and structured generation.
- **RAG Helper:** Auto-chunk and embed content on model save.
