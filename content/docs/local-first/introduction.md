# Local-First & Cross-Platform

> **Note:** This feature is planned for a future release and is not yet implemented.

OmniSvelte will extend its reach beyond traditional web apps to local-first and native mobile/desktop platforms.

## Planned Features

- **`defineLocalModel()`:** Dexie-backed local storage with sync-when-online and conflict resolution.
- **Tauri / Capacitor Scaffolding:** `omni add tauri` and `omni add capacitor` commands to easily package web apps for desktop and mobile.
